# 시편 149

- 1: 할렐루야. 여러분은 여호와께 노래하세요, 새 노래로, 한결같은 사람들의 모임에서 여호와께 드리는 찬양으로.
- 2: 이스라엘은 즐거워해야지요, 자신들을 만드신 분을 생각하면서! 시온 사람들은 기뻐해야지요, 자신들의 임금을 생각하면서!
- 3: 그들은 여호와의 이름을 찬양해야지요, 춤추면서! 북 치고 기타 치며 여호와께 찬송해야지요!
- 4: 여호와가 자신의 백성을 좋아하시니까요. 겸손한 사람들을 구원하심으로 영광스럽게 해 주시니까요.
- 5: 한결같은 사람들은 영광을 누리면서 기뻐해야지요! 잠자리에서도 기뻐 외쳐야지요!
- 6: 하나님 높이는 소리를 목청껏 내야지요, 손에는 날 선 칼을 들고서요.
- 7: 민족들에 대한 보복을, 겨레들에 대한 처벌을 집행하도록.
- 8: 그들의 임금들을 사슬로, 높은 사람들을 쇠사슬로 묶도록.
- 9: 적어 둔 판결대로 그들에게 하도록. 이것이 여호와께 한결같은 모든 사람이 누리는 명예이지요. 할렐루야.
